# types

A place for types

## Add to a project

With NPM:

```
$ npm install -D @guardian/types
```

With Yarn:

```
$ yarn add -D @guardian/types
```

## Development

### Run the tests

```
$ npm test
```

### Publish to NPM

Bump the version in `package.json`.

Publish the package:

```
$ npm publish --access public
```
